Relationships

Changes

Class A changes class B when class A directly accesses or changes and element of B.

Composes

Class A composes class B when class B must consume some number of instances of class A in order to be instantiated.

Commands

Class A commands class B when class B queries class A in order to determine which action to take.

Creates

Class A creates class B when class A instantiates a new copy of class B.

Uses

Class A uses class B when class A calls public methods of class B in order to query the state of class B, and then uses that state information to influence the behavior of class A.
